# rest-countries-api-python

This project uses Python and the REST Countries API to fetch and display information about a country.

## Features

- Search for a country by name
- Display:
  - Country name
  - Capital
  - Region
  - Population
  - Languages
  - Currencies

 ## Technologies Used

 - Python
 - Requests library
 - REST Countries API
